The Dialog service will keep track of dialogs that have been opened. When the dialog is closed, the service will destroy the reference.

From the service, all dialogs can be closed by calling closeAllDialogs().

Example Use:

If your application includes a session idle monitoring service, the dialog service can be invoked to close all active dialogs when the idle service logs out the user. This ensures that there is no inadvertent access due to an open dialog when the session terminates.
